# Penko Typing Roadmap

**Current Status:** Beta v0.1.0 - Core Gameplay Complete
**Goal:** Best free typing game for learning non-Latin keyboard layouts

This roadmap outlines future enhancements to make Penko Typing the most comprehensive typing tutor for international keyboards.

---

## Current Features (Complete)

- [x] 13 keyboard layouts (English, Spanish, French, German, Korean, Russian, Japanese, Greek, Arabic, Hebrew, Thai, Hindi, Ukrainian)
- [x] Retro arcade falling-word gameplay
- [x] Finger mapping visualization with color-coded keys
- [x] Real-time WPM tracking
- [x] Accuracy metrics
- [x] Local leaderboard system
- [x] Multilingual UI (13 languages)
- [x] Toggle hand visualization on/off
- [x] PWA offline support
- [x] Free practice mode
- [x] Manual/Tutorial mode

## Development Principles

### Always Maintain
1. **Free Forever** - No subscriptions, no ads, no premium tiers
2. **Privacy-First** - All data local, no tracking, no analytics without consent
3. **Offline-First** - Full functionality without internet
4. **Open Source** - GPL v3 licensed, community-driven
5. **Accessibility** - Usable by everyone, regardless of ability

### Technology Stack
- Frontend: React 19, TypeScript, Tailwind CSS
- Build: Vite
- Storage: Browser localStorage
- PWA: Service Worker, manifest.json
- Hosting: GitHub Pages (static)
